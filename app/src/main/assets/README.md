# Assets 폴더

이 폴더에 Supertonic TTS 모델 파일을 배치하세요.

## 필요한 파일

- `supertonic_model.onnx`: Supertonic TTS 모델 파일
  - Supertonic 저장소에서 다운로드하거나
  - Hugging Face에서 제공하는 모델을 사용

## 모델 다운로드

- GitHub: https://github.com/supertone-inc/supertonic
- Hugging Face: https://huggingface.co/spaces/Supertone/supertonic
